# Dashboard

The Dashboard is your central command center for monitoring and managing your lead generation
activities in Ettutu V2.

<!-- Screenshot: Dashboard overview showing all widgets and metrics -->

---

## Overview

When you log in, the Dashboard provides an at-a-glance view of:

- **Key Performance Metrics** - Total records, active leads, search sessions
- **Data Completeness** - Progress indicators for enriched data
- **Lead Quality Distribution** - Visual breakdown of lead scoring
- **Pipeline Status** - Overview of leads in each CRM stage
- **Recent Activity** - Your latest search sessions

---

## Key Metrics

| Metric              | Description                                       |
| ------------------- | ------------------------------------------------- |
| **Total Records**   | All business records in your database             |
| **Active Leads**    | Records currently in the sales pipeline           |
| **Active Searches** | Running or recent Google Places search sessions   |
| **Contact Data %**  | Percentage of records with extracted contact info |

<!-- Screenshot: Key metrics cards at top of dashboard -->

---

## Data Completeness

Progress bars show how much contact data has been extracted:

- **Emails Found** - Percentage of records with email addresses
- **Websites Crawled** - Records that have been processed by crawlers
- **Phone Numbers** - Records with phone contact information

This helps you identify where to focus crawling and analysis efforts.

---

## Lead Quality Distribution

A visual chart showing the breakdown of your leads:

| Quality    | Description                                           |
| ---------- | ----------------------------------------------------- |
| **High**   | Strong potential - complete data, clear business need |
| **Medium** | Good potential - partial data or unclear fit          |
| **Low**    | Weak potential - minimal data or poor fit             |

Quality scores are assigned by the AI Lead Analyzer based on crawled website content.

---

## Pipeline Overview

See how many leads are in each stage:

1. **New** - Fresh leads not yet contacted
1. **Contacted** - Initial outreach made
1. **Qualified** - Confirmed interest and fit
1. **Proposal** - Sent pricing/proposal
1. **Negotiation** - Active discussions
1. **Closed** - Won or lost deals

Click any stage to jump directly to the Pipeline view filtered by that stage.

---

## Recent Searches

Shows your last 5 search sessions with:

- Search query and location
- Number of results found
- Crawl completion percentage
- Quick action buttons

Click a session to view its records or start crawling.

---

## Quick Actions

Buttons for common tasks:

| Action            | Description                      |
| ----------------- | -------------------------------- |
| **New Search**    | Start a new Google Places search |
| **View Records**  | Go to the Records database       |
| **View Contacts** | See your verified contacts       |

---

## System Status

Bottom indicator showing:

- **Online/Offline** status
- **Application version**
- **Worker status** (if jobs are processing)

---

## Tips for Using the Dashboard

1. **Start your day here** - Review metrics and activity before diving into work
1. **Monitor data completeness** - Low percentages indicate records needing crawling
1. **Watch lead quality** - Adjust search queries if quality is consistently low
1. **Check pipeline** - Ensure leads aren't stuck in early stages
